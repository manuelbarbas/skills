# CTX Compiler and Gas Rules

Use these rules whenever the request involves BITE Phase II / CTX.

## Rules

- Apply compiler and EVM version requirements from `references/phase-2-ctx.md` exactly.
- Include CTX gas/payment requirements in implementation instructions.
- Do not omit required callbacks/interfaces for CTX contracts.
- If compiler settings conflict with user project settings, call out the mismatch and provide exact remediation.
